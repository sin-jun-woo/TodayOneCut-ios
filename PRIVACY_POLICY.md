# 오늘의 한 컷 개인정보 처리방침

**Privacy Policy – 오늘의 한 컷 (Today's One Cut)**  
**최종 업데이트: 2026-03-09**

**App Store Guideline 5.1.1 & 5.1.2 준수:** 본 앱은 사용자 데이터를 **서버로 전송하지 않으며**, **제3자에게 제공·판매·공유하지 않습니다.** 모든 데이터는 **사용자 기기 내부에만** 수집·저장·이용됩니다. (Data Collection and Storage · Data Use and Sharing compliant.)

오늘의 한 컷("본 앱")은 사용자의 개인정보를 소중히 하며, 수집·저장·이용·제공에 관한 사항을 아래와 같이 안내합니다.

---

## 1. 수집하는 정보 (Data Collection) — 5.1.1 준수

본 앱은 **서버에 어떠한 데이터도 전송하지 않습니다.**  
모든 데이터는 **사용자 기기 내부에만** 수집·저장됩니다.

### 1.1 카메라·사진 데이터 (Camera & Photo Data)

| 항목 | 내용 |
|------|------|
| **수집 시점** | 사용자가 "사진으로 기록하기"를 선택하고 카메라로 촬영하거나, 사진 라이브러리에서 이미지를 선택할 때만 수집됩니다. |
| **수집 내용** | 촬영하거나 선택한 **사진 이미지 파일**만 수집합니다. 카메라를 통한 실시간 영상 스트림은 저장하지 않습니다. |
| **저장 위치** | **기기 내부 저장소**에만 저장됩니다. (iOS Core Data 및 앱 샌드박스 내 파일) |
| **전송 여부** | **없음.** 인터넷 또는 외부 서버로 전송되지 않습니다. |

- 카메라 권한: 사진 촬영 시에만 사용되며, 사용자가 "허용"한 경우에만 사용됩니다.
- 사진 라이브러리 권한: 사용자가 갤러리에서 사진을 선택할 때만 사용되며, 선택한 사진만 앱 내 기록으로 저장됩니다.

### 1.2 텍스트 데이터 (Text / User-created content)

| 항목 | 내용 |
|------|------|
| **수집 시점** | 사용자가 "오늘의 한 컷" 기록에 텍스트(메모 또는 글)를 입력할 때만 수집됩니다. |
| **수집 내용** | 사용자가 직접 입력한 **텍스트 내용**만 수집합니다. |
| **저장 위치** | **기기 내부 저장소**에만 저장됩니다. (Core Data) |
| **전송 여부** | **없음.** 외부 서버로 전송되지 않습니다. |

### 1.3 위치 정보 (Location Data)

| 항목 | 내용 |
|------|------|
| **수집 시점** | 사용자가 설정에서 "위치 정보 사용"을 켠 경우에만, 기록 저장 시점에 한 번 수집됩니다. |
| **수집 내용** | 해당 시점의 **위치 좌표 및 역지오코딩된 장소 이름**만 기록에 첨부됩니다. |
| **저장 위치** | **기기 내부 저장소**에만 저장됩니다. (Core Data) |
| **전송 여부** | **없음.** 외부 서버로 전송되지 않습니다. |

위치 권한은 선택 사항이며, 사용자가 거부해도 앱의 핵심 기능(사진·텍스트 기록)은 이용 가능합니다.

### 1.4 기타 데이터

- **설정 정보**: 테마, 알림 on/off, 위치 사용 on/off 등은 **기기 내부에만** 저장됩니다.
- **알림 권한**: 일일 리마인더 알림을 보내기 위한 용도로만 사용되며, 수집된 데이터를 서버로 보내지 않습니다.

## 2. 데이터의 이용 (Data Use) — 5.1.2 준수

수집된 데이터는 **아래 목적으로만** 이용됩니다.

- **기록 보기·편집·삭제**: 사용자가 작성한 "오늘의 한 컷" 기록을 앱 내에서만 표시·관리하는 데 사용됩니다.
- **앱 기능 제공**: 사진·텍스트·날짜·위치를 사용자 본인의 기록으로만 표시하며, 이 목적 외로 사용하지 않습니다.

본 앱은 수집된 데이터를 다음 목적으로 **이용하지 않습니다.**

- **광고** 목적으로 이용하지 않습니다.
- **분석·통계·행동 추적** 목적으로 이용하지 않습니다.
- **마케팅** 목적으로 이용하지 않습니다.
- **제3자에게 제공·판매·공유**하지 않습니다.

---

## 3. 데이터 저장 및 보관 (Storage & Retention)

- **저장 위치**: 모든 데이터는 **사용자 기기의 로컬 저장소**(Core Data 및 앱 샌드박스)에만 저장됩니다.
- **클라우드·서버**: 클라우드 동기화나 외부 서버 저장 기능이 없으며, 데이터를 서버로 보내지 않습니다.
- **보관 기간**: 사용자가 앱을 삭제할 때까지 기기 내에 보관되며, **앱 삭제 시 해당 기기에서 저장된 데이터는 함께 삭제**됩니다.

---

## 4. 데이터 보안 (Security)

- iOS가 제공하는 **기기 내 저장소 및 암호화** 환경을 사용합니다.
- 데이터는 **앱 샌드박스** 내에만 저장되어, 다른 앱 또는 외부에서 직접 접근할 수 없습니다.

---

## 5. 제3자 제공 및 공유 (Sharing) — 5.1.2 준수

- 본 앱은 수집한 **어떠한 데이터도 제3자에게 제공·판매·공유하지 않습니다.**
- 광고 SDK, 분석 SDK, 소셜 로그인 등 **제3자 서비스를 사용하지 않으며**, 따라서 제3자에게 데이터가 전달되는 경로가 없습니다.
